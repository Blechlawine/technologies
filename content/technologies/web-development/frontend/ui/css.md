## Tailwindcss

[Website (https://tailwindcss.com/)](https://tailwindcss.com/)

CSS Utility classes

| Pros                                                 | Cons                                             |
| ---------------------------------------------------- | ------------------------------------------------ |
| Popular                                              | You get very long class definitions in your HTML |
| Extensions for most editors                          |                                                  |
| Very flexible                                        |                                                  |
| In combination with DaisyUI great UI building blocks |                                                  |
