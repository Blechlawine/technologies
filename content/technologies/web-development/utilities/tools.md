| ----------------------------------------------------- | ---- |
| Slides are Markdown and/or Vue-components             |      |
| Customizable themeing                                 |      |
| Export to PDF, images, etc.                           |      |
| Interactive coding in the presentation                |      |
| Presenter mode to trigger slides on different monitor |      |
| Animations                                            |      |
